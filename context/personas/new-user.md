# Persona: New User (First Install, First Launch)

> User installs app for the first time, opens immediately, has no prior context.

## What to Test Across All Modules

| Module | Persona-Specific Focus |
|--------|----------------------|
| **Onboarding** | Full flow: all screens, questions, permissions. Time to first value. Clarity of core feature |
| **Homepage & IA** | First impression of navigation. Can user identify core feature? Cognitive load on first screen |
| **Discovery & Search** | Does user know what to search for? Is browse path obvious without prior context? |
| **Personalization** | Are JTBD questions asked? Do answers visibly change the experience? |
| **Content Consumption** | First content interaction: how many taps? Is feedback immediate? |
| **Engagement & Retention** | Is there a hook in the first session? Reason to return tomorrow? |
| **Monetization** | When does first paywall appear relative to first value? Is it before or after? |
| **AI Integration** | Does AI help onboarding? Is AI feature explained or hidden? |
| **Cognitive Load** | How many choices on first screen? Is primary CTA obvious? |
| **Persona-Intent** | Does the app understand why this user installed it? Does it ask? |

## Key Metrics for This Persona

- **Primary**: Activation rate (% completing onboarding and reaching first value)
- **Time to first value** (seconds from app open)
- **Steps to first value** (tap count)
- **Permission friction** (count of requests before value)
- **Paywall timing** (before/during/after first value moment)

## Scoring Focus

Score this persona higher when: time to value <60s, personalization present, paywall after value.
Score lower when: paywall before value, no personalization, >5 minutes to first interaction.
