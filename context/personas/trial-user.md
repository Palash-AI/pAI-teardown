# Persona: Trial User (Active Trial, Before Expiry)

> User started trial, is experiencing premium features mid-trial.

## What to Test Across All Modules

| Module | Persona-Specific Focus |
|--------|----------------------|
| **Onboarding** | Is there onboarding specific to premium features? |
| **Homepage & IA** | Are premium features highlighted/badged? Is trial status visible? |
| **Discovery & Search** | Can user discover and access all premium content? |
| **Personalization** | Is premium personalization noticeably better than free? |
| **Content Consumption** | Premium content quality vs free — is the difference obvious? |
| **Engagement & Retention** | Does trial experience build habit that's hard to break? |
| **Monetization** | Trial status indicator? Days remaining? Conversion messaging? |
| **AI Integration** | AI features unlocked in trial? Are they showcased? |
| **Cognitive Load** | Is premium UI cleaner (no ads, no paywalls)? |
| **Persona-Intent** | Does premium tier better serve the user's actual job? |

## Key Metrics

- **Primary**: Trial-to-paid conversion rate signal
- **Premium feature discovery** (did user try key premium features?)
- **Trial status visibility** (does user know when trial ends?)
- **Conversion messaging timing** and quality

## Scoring Focus

Score higher when: trial onboards to premium features, trial status is clear, value difference is obvious.
Score lower when: trial experience = free experience, no guidance, no conversion messaging until last day.
