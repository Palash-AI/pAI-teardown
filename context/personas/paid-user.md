# Persona: Paid User (Active Subscription)

> User has paid for subscription. Experiencing the full premium product.

## What to Test Across All Modules

| Module | Persona-Specific Focus |
|--------|----------------------|
| **Onboarding** | Is there premium onboarding? Purchase celebration? |
| **Homepage & IA** | Premium features clearly marked? Post-purchase experience feels premium? |
| **Discovery & Search** | Full content library accessible? Premium filters/features? |
| **Personalization** | Premium-tier personalization (AI recommendations, adaptive learning)? |
| **Content Consumption** | Ad-free? Downloads? Offline mode? Premium content quality? |
| **Engagement & Retention** | Value reinforcement? Does app remind user what they're getting? |
| **Monetization** | Renewal transparency? Billing date visible? Easy cancellation? |
| **AI Integration** | Premium AI features working? Noticeably better than free? |
| **Cognitive Load** | Cleaner UI without upsell interrupts? |
| **Persona-Intent** | Does premium tier fully serve the user's job? Any remaining gaps? |

## Key Metrics

- **Primary**: Feature adoption within paid tier / renewal rate signals
- **Value reinforcement** (does app acknowledge premium status?)
- **Renewal transparency** (billing date, amount, cancellation path)
- **Premium feature clarity** (does user know all they have access to?)

## Scoring Focus

Score higher when: premium features clearly shown, purchase celebrated, renewal transparent, easy cancel.
Score lower when: premium = free with no visible difference, no celebration, hidden cancellation.
