# Persona: Free Returning User (Day 2+, No Subscription)

> User completed onboarding yesterday/previously, opens app today as a free user.

## What to Test Across All Modules

| Module | Persona-Specific Focus |
|--------|----------------------|
| **Onboarding** | Is onboarding repeated or skipped? Does app remember prior session? |
| **Homepage & IA** | Is content personalized from prior visit? Any "welcome back" messaging? |
| **Discovery & Search** | Does search reflect prior interests? Recent/saved content accessible? |
| **Personalization** | Are prior preferences retained? Does content change based on history? |
| **Content Consumption** | Resume from where user left off? Progress saved? |
| **Engagement & Retention** | Streaks, notifications, re-engagement hooks present? |
| **Monetization** | How many paywalls per session? Soft vs hard? Can user continue free? |
| **AI Integration** | Does AI reference prior usage? Personalized suggestions? |
| **Cognitive Load** | Is the second visit simpler than the first? Less guidance needed? |
| **Persona-Intent** | Does app remember the user's job? Serve it proactively? |

## Key Metrics

- **Primary**: Day-1 retention / session frequency
- **Paywall frequency** per session (count soft + hard)
- **Personalization retention** (do prior preferences persist?)
- **Content progress** continuity

## Scoring Focus

Score higher when: prior context is remembered, paywalls are soft, free content is genuinely useful.
Score lower when: onboarding repeats, paywall fatigue (>3/session), no memory of prior visit.
