## Critical Design Deltas

| Area | Global Default | India Reality |
|------|---------------|--------------|
| **Onboarding language** | English | Hindi/Hinglish for Tier 2-3 or lose 60% |
| **Data usage** | Unlimited assumed | Metered. Offline/low-data mode is Must-Be for Tier 2-3 |
| **Payment method** | Credit card | UPI is default (80%+ of digital payments). No UPI = no Tier 2-3 revenue |
| **Price sensitivity** | $10-20/mo normal | ₹99-199/mo is ceiling for most. Competes with mobile recharge, not Netflix |
| **Annual vs monthly** | Monthly default | Annual preferred (feels like a "deal"). Minimum 25-30% discount expected |
| **₹1 trial** | Universal positive | Brand-dependent. Trusted brands = irresistible. Unknown = "scam" signal |
| **Share mechanics** | Social media share | WhatsApp-first (pre-filled message + deep link). "Tell 3 friends" > "post to social" |
| **Trust signals** | Reviews sufficient | Need: ₹ pricing, known payment gateway logos, Hindi copy, user count |
| **Content format** | Text + video | Short video (2-5 min) dominates. Reels/Shorts trained attention spans |
| **Family dynamics** | Individual user | Phone sharing common Tier 2-3. Family plans resonate. Privacy implications |
| **Notification tolerance** | Low (2-3/day) | Higher but WhatsApp-calibrated. In-app > push for Tier 2-3 |
| **Competitor benchmark** | Global leaders | Unacademy, PhysicsWallah, Byju's, Josh (short video), ShareChat |
